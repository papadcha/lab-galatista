v1.1.20 — 2026-07-03  [ΚΡΙΣΙΜΟ]
----------------------------------------------------------------
Update-checker κολλημένος σε παλιά έκδοση (HTTP cache):
- Bug: το ενσωματωμένο update-checker δεν έδειχνε ΠΟΤΕ νέα έκδοση.
  Αιτία: το Electron net.request() cache-άρει την απάντηση του GitHub
  API στον δίσκο, και δεν την ανανέωνε σωστά μεταξύ επανεκκινήσεων —
  η πρώτη ποτέ απάντηση έμενε κολλημένη επ' άπειρον (πολύ πέρα από τα
  δηλωμένα 60s cache του GitHub).
- Fix: cache-busting query param σε κάθε request — μοναδικό URL,
  πάντα fresh fetch. Επαληθεύτηκε: 2 διαδοχικές εκκινήσεις έφεραν
  σωστά τη νέα έκδοση ακόμα και με προϋπάρχον cache.
- ΣΗΜΑΝΤΙΚΟ: όποια εγκατάσταση έχει ήδη «κολλημένο» cache από πριν
  ΔΕΝ θα δει το update-checker να δουλεύει ούτε για αυτό το ίδιο το
  v1.1.20 — χρειάζεται ΜΙΑ τελευταία χειροκίνητη εγκατάσταση για να
  σπάσει ο κύκλος. Μετά, θα δουλεύει κανονικά.

ΣΗΜΕΙΩΣΗ ΣΥΝΤΗΡΗΣΗΣ (για τον προγραμματιστή)
----------------------------------------------------------------
Αυτό το αρχείο μπαίνει bundled μέσα στον installer (package.json →
build.files) και διαβάζεται από την εφαρμογή μέσω IPC
('get-version-history', main.js) για να εμφανιστεί σε modal όταν ο
χρήστης πατήσει τον αριθμό έκδοσης στο sidebar footer
(#sidebar-version, main-app.js). Ενημέρωσέ το σε κάθε version bump,
μαζί με το CHANGELOG.md (τεχνικό, πλήρες) και το ξεχωριστό αρχείο
TODOLIST.md (to-do list, μόνο Desktop).
