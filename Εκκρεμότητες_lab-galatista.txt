v1.1.12 — 2026-07-02
----------------------------------------------------------------
Δελτίο Αποτελεσμάτων (Εκθέσεις):
- Διαγνωστικό fix: το toast σφάλματος στο κουμπί Αποθήκευση έδειχνε

v1.1.11 — 2026-07-02
----------------------------------------------------------------
Frameless window:
- Αφαίρεση native Windows title bar και του default Electron menu
  (File/Edit/View) — αντικαταστάθηκαν με custom titlebar (drag region
  + κουμπιά ελαχιστοποίησης/μεγιστοποίησης/κλεισίματος) που ακολουθεί
  το θέμα της εφαρμογής. Έκλεισε το GitHub issue #5.

Βιβλιοθήκη Εγγράφων:
- Sync μεταδεδομένων βιβλιοθήκης εγγράφων μεταξύ εγκαταστάσεων
  (διαχειριστής/χειριστής). Τα αρχεία ανέβαιναν ήδη σε κοινό cloud
  path ένα-ένα χειροκίνητα, αλλά οι εγγραφές που τα περιγράφουν ήταν
  μόνο τοπικές — έγγραφο που πρόσθετε το ένα μηχάνημα ήταν αόρατο στο
  άλλο. Additions-only sync (χωρίς διαγραφές) μέσω JSON manifest ανά
  μηχάνημα. Νέο κουμπί "Sync Βιβλιοθήκης" στις Ρυθμίσεις (μέσα στο
  cloud sync panel — εμφανίζεται μόνο αφού έχει ρυθμιστεί cloud
  remote + path). Έκλεισε το GitHub issue #4.
- Εκτός εμβέλειας (σκόπιμα): διάδοση επεξεργασίας/διαγραφής — χρειάζεται
  soft-delete tombstones.


v1.1.10 — 2026-07-01
----------------------------------------------------------------
Υποπερίοδοι CE:
- Νέο: επεξεργάσιμη ημερομηνία έναρξης υποπεριόδου (Ρυθμίσεις →
  Εργαστήριο → Επεξεργασία Υποπεριόδου). Όταν αλλάζει, όλα τα δείγματα
  της CE period επανα-αξιολογούνται αυτόματα και αλλάζουν υποπερίοδο
  ανάλογα — λύνει την περίπτωση όπου η υποπερίοδος ξεκίνησε μία μέρα
  αργότερα από τα πραγματικά δείγματα.

Γραμματοσειρές — πλήρης έλεγχος project:
- Fix: κωδικός δείγματος στις Εκθέσεις είχε γενική system monospace
  αντί για το θέμα της εφαρμογής (IBM Plex Sans/.sample-code).
- Ίδιο μοτίβο (font-family:monospace γενικό) διορθώθηκε σε ~20 σημεία
  σε 9 αρχεία (main-app.js, reports, settings, history, samples,
  library, main.css).
- Ευθυγραμμίστηκαν και οι οδηγοί δοκιμών (SE/MB/ΚΚΜ-ΠΛΚ) + SVG
  diagrams σε IBM Plex Sans, με τοπικά bundled fonts (χωρίς εξάρτηση
  από Google Fonts/internet).


v1.1.9 — 2026-07-01
----------------------------------------------------------------
Backup & Cloud Sync:
- Τοπικά backups: VACUUM INTO αντί ωμού αντιγράφου (δεν χάνει
  δεδομένα από το -wal), filename με ώρα (όχι μόνο ημέρα), hash
  dedup — κρατά τα 7 πιο πρόσφατα distinct.
- Cloud sync (upload + "Επαναφορά από Cloud"): rclone sync→copy και
  στις δύο κατευθύνσεις — δύο εγκαταστάσεις (διαχειριστής/χειριστής)
  στο ίδιο cloud remote δεν σβήνουν πια η μία τα backups/PDF της
  άλλης.
- Νέο: προειδοποιητικό toast στο startup όταν ο τοπικός φάκελος
  δεδομένων διαφέρει από της ενεργής CE period στη βάση.
- Fix: getDbPath() σε dev mode "κολλούσε" σε παλιό αρχείο βάσης.
- Αφαίρεση ξεπερασμένου setup_db.sh, fix idempotency seed_data.py.

Δελτίο Αποτελεσμάτων (Εκθέσεις):
- Fix: όνομα προϊόντος έδειχνε σπασμένο ("ΑΜΜΟΣmm") αντί "ΑΜΜΟΣ 0/4".
- Fix: λογότυπο δεν εμφανιζόταν (λάθος relative path).
- Fix: επωνυμία εργαστηρίου ήταν hardcoded αντί δυναμική.
- Fix: προεπισκόπηση με άσχετη γραμματοσειρά (Arial/Times New Roman)
  — τώρα ακολουθεί δυναμικά τη "Γραμματοσειρά PDF" των Ρυθμίσεων.
